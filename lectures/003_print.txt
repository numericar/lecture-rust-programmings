print!
    เป็นการแสดงข้อความไปที่ console แต่ไม่เพิ่มบรรทัดใหม่
println!
    เป็นการแสดงข้อความไปที่ console และทำการเพิ่มบรรทัดใหม่
eprintln!
    คล้ายกับ println! แต่ใช้สำหรับแสดงข้อผิดพลาดออกมาที่ console
    โดยการทำงานจะไม่ใช่รูปแบบ Buffer จึงจะสามารถแสดงผลได้ก่อน print อื่น ๆ
format!
    เป็นการสร้าง string ใหม่ตามรูปแบบที่กำหนด โดยจะใช้สำหรับ string ที่ต้องการแสดงข้อมูลจากตัวแปรด้วย
write!
    คล้ายกับ format! แต่จะเขียนไปที่ string เดิมแทนที่จะสร้าง string ใหม่
